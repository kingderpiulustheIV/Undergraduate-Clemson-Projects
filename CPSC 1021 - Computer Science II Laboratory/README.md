# CPSC 1021 - Computer Science II Laboratory

## Course Description
Laboratory course accompanying CPSC 1020. Provides hands-on experience with C++ programming, data structures, algorithms, and software development practices. Focus on practical implementation and problem-solving through weekly lab assignments.

## Learning Objectives
- Apply object-oriented programming principles in C++
- Implement and test fundamental data structures
- Practice file I/O and data processing
- Develop debugging and testing skills
- Collaborate effectively in team programming environments
- Master C++ memory management and pointers

## Lab Assignments

### Lab 1: Bike-to-Campus Reward Points
Calculate and manage reward points for campus bike commuters using basic C++ I/O and calculations.

### Lab 1: Retirement Calculator
Financial calculator implementing compound interest and retirement planning algorithms.

### Lab 2: Makefile
Learn build automation and dependency management using Makefiles.

### Lab 2: Patient Charges
Medical billing system demonstrating procedural programming and data processing.

### Lab 3: Areas Calculation
Geometric calculations using functions and basic algorithms.

### Lab 4: Patient Charges with Classes
Refactor of patient charges system using object-oriented design and C++ classes.

### Lab 5: Parking Permits
Parking permit management system implementing class hierarchies and data validation.

### Lab 6: Date Class with IO
Custom date class implementation with operator overloading and stream I/O.

### Lab 7: Corporate Sales
Data analysis program processing sales information across multiple datasets.

### Lab 7: Palindrome Testing
String manipulation and algorithm implementation for palindrome detection.

### Lab 7: Rational Arithmetic
Rational number class with arithmetic operations and operator overloading.

### Lab 7: String Encryption
Cryptography basics implementing encryption/decryption algorithms.

### Lab 8: Area Calculation using Inheritance and Polymorphism
Shape hierarchy demonstrating inheritance and polymorphic behavior.

### Lab 9: Area Calculation using Inheritance, Polymorphism & Templates
Advanced template programming combined with inheritance for generic shape calculations.

### Lab 9: Sequence Accumulation
Template-based sequence processing and accumulation algorithms.
