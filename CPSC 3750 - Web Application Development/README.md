# CPSC 3750 - Web Application Development

## Course Description
Comprehensive web development course covering front-end and back-end technologies. Students learn to build full-stack web applications using HTML, CSS, JavaScript, PHP, and databases. Topics include responsive design, client-server architecture, AJAX, session management, and modern web frameworks.

## Learning Objectives
- Master HTML5, CSS3, and responsive web design
- Develop interactive web applications using JavaScript and jQuery
- Implement server-side programming with PHP
- Integrate relational databases (SQL) with web applications
- Use AJAX for asynchronous communication
- Apply modern web frameworks and templating engines
- Understand web security and session management
- Practice usability testing and user experience design
- Deploy and maintain web applications

## Projects and Assignments

### HTML & CSS Fundamentals
- **Demo HTML & CSS**: Introduction to HTML structure and CSS styling
- **Basic Fixed Fluid Hybrid Layout**: Responsive layout techniques combining fixed and fluid designs

### JavaScript & Client-Side Programming
- **JavaScript Events**: Event handling and DOM manipulation
- **JavaScript Sort**: Array manipulation and sorting algorithms
- **Keypress**: Keyboard event handling
- **Moving Buttons**: Dynamic UI elements and animations
- **Card Object**: Object-oriented JavaScript programming

### jQuery & AJAX
- **jQuery**: jQuery library for simplified DOM manipulation
- **AJAX & Handlebars**: Asynchronous requests with Handlebars templating
- **AJAX & PHP**: Server communication using AJAX and PHP backend

### PHP & Server-Side Development
- **PHP Helloworld**: Introduction to PHP programming
- **PHP File I/O App**: File operations and data persistence
- **Common NavBar**: Reusable navigation components with PHP includes

### Database Integration
- **Integrate with DB**: Database connectivity and CRUD operations

### Group Projects
- **Group Audio**: Audio manipulation web application
- **Group Events**: Event management system
- **Group Forms**: Form validation and processing
- **Group Hangman**: Interactive game implementation
- **Group More CSS**: Advanced CSS techniques and styling
- **Group Sessions**: Session management and user authentication
- **Group Zipcode Distance**: Geographic calculations using web services

### Major Projects
- **Project 2: Collect App Phase 1**: Database-driven collection management application of pokemon essentially a Pokedex. 
- **Program Exam #1**: Practical assessment of web development fundamentals
- **Program Exam 2**: Advanced web development assessment

### Advanced Topics
- **Audio Changer**: Audio processing and manipulation
- **Usability Testing**: User experience evaluation and testing methodologies

## Technology Stack

### Front-End Technologies
- **HTML5**: Markup language for web content
- **CSS3**: Styling and responsive design
- **JavaScript (ES6+)**: Client-side programming
- **jQuery**: JavaScript library for DOM manipulation
- **Handlebars.js**: Templating engine

### Back-End Technologies
- **PHP**: Server-side scripting language
- **MySQL/phpMyAdmin**: Relational database management

### Web Development Concepts
- AJAX (Asynchronous JavaScript and XML)
- RESTful API design
- Session management and cookies
- Form validation and processing
- Responsive web design
- Client-server architecture

### Development Tools
- Web browser developer tools
- Text editor or IDE (VS Code)
- Git version control
- Database management tools (phpMyAdmin, mySQL Workbench)

### Framework and Libraries
- jQuery
- Handlebars.js

### Environment
- Apache web server
- PHP runtime environment
- MySQL database server
- Local development environment (XAMPP/WAMP/MAMP)

## Setup and Requirements
- Web server (Apache recommended)
- PHP 7.0 or later
- MySQL/MariaDB database server
- Modern web browser (Chrome, Firefox, Edge)
- Text editor or IDE
- XAMPP/WAMP/MAMP for local development
- Basic understanding of HTTP protocol
- Knowledge of command-line tools (optional but helpful)

## Work Type
This class includes both **individual and group-based work** using HTML, CSS, JavaScript, PHP, and MySQL.
