# CPSC 2311 - Introduction to Computer Organization Lab

## Course Description
Laboratory course providing hands-on experience with low-level programming, computer architecture, and system-level programming. Students work with C and Assembly language to understand how software interacts with hardware, including memory management, processor architecture, and system calls.

## Learning Objectives
- Understand computer architecture and organization
- Master low-level programming in C and Assembly
- Learn memory management and pointer manipulation
- Understand the compilation process and linking
- Work with system calls and operating system interfaces
- Debug and optimize low-level code
- Understand data representation (integers, floats, characters)
- Learn assembly language programming and instruction sets

## Technology Stack

### Programming Languages
- **C**: Primary language for system-level programming
- **Assembly Language**: Low-level programming and instruction set architecture (x86/x86-64)

### Development Tools
- GCC compiler
- GDB debugger
- Make build system
- Assembler (NASM, GAS, or MASM)
- Valgrind for memory debugging
- Objdump and nm for binary analysis

### Concepts Covered
- Memory Management (Stack, Heap)
- Pointers and Dynamic Allocation
- System Calls
- Data Representation
- CPU Architecture
- Assembly Programming
- Linking and Loading
- Performance Optimization

- Linux/Unix development environment

- System-level programming interfaces
