# CPSC 3720 - Introduction to Software Engineering

## Course Description
Advanced software engineering course focusing on cloud computing, serverless architecture, API development, and modern software deployment practices. Students learn to design and implement RESTful APIs, work with cloud services (AWS), and use industry-standard tools for API development and testing.

## Learning Objectives
- Design and implement RESTful APIs following industry standards
- Master cloud computing concepts and AWS services
- Develop serverless applications using AWS Lambda
- Create API schemas using YAML/OpenAPI specification
- Test and deploy APIs using professional tools (Postman)
- Understand microservices architecture
- Practice collaborative software development in teams
- Learn API security and authentication best practices

## Projects and Assignments

### Main Project: RESTful API Development
Design and implementation of a comprehensive HTTPS RESTful API deployed on AWS infrastructure. The project involves:
- API design and schema definition
- AWS Lambda function development
- YAML schema creation for API specification
- JSON request and response body handling
- API testing and deployment using Postman
- Cloud infrastructure management

## Technology Stack

### Cloud Platform
- **Amazon Web Services (AWS)**: Cloud infrastructure platform
- **AWS Lambda**: Serverless compute service for running API functions
- **AWS API Gateway**: API management and deployment

### Programming Language
- **Node.js**: Primary runtime for Lambda functions
- **JavaScript**: Server-side programming language

### API Development
- **RESTful API Architecture**: Stateless API design pattern
- **HTTPS**: Secure communication protocol
- **YAML**: Schema definition and configuration
- **JSON**: Data interchange format for requests and responses
- **OpenAPI/Swagger**: API specification standards

### Development and Testing Tools
- **Postman**: API testing and deployment platform
- **AWS Console**: Cloud resource management
- **Git**: Version control
- **Node Package Manager (npm)**: Dependency management

### Documentation
- API documentation using OpenAPI/YAML schemas
- Postman collections for API testing

### Environment
- Cloud-based deployment (AWS)
- Local development environment for Node.js
- Web-based API testing (Postman)
- Windows 10 x86-64

## Setup and Requirements
- AWS account with Lambda and API Gateway access
- Node.js and npm installed
- Postman account for API testing
- Understanding of HTTP protocols and REST principles
- Knowledge of JSON data structures
- Basic cloud computing concepts
