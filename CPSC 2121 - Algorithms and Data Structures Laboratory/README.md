# CPSC 2121 - Algorithms and Data Structures Laboratory

## Course Description
Laboratory course accompanying CPSC 2120. Provides hands-on implementation experience with advanced data structures and algorithms through weekly programming labs. Emphasizes practical application of theoretical concepts, algorithm optimization, and problem-solving.

## Learning Objectives
- Implement complex data structures from scratch
- Apply algorithmic techniques to real-world problems
- Practice recursion and backtracking algorithms
- Master graph algorithms and tree traversals
- Optimize code for performance and efficiency
- Develop skills in algorithm analysis and testing

## Lab Assignments

### Lab 1: Linked List Based Stacks and Queues
Implement stack and queue abstract data types using linked list underlying structure.

### Lab 2: Infix to Postfix Conversion
Expression parsing and conversion using stack-based algorithm for mathematical notation transformation.

### Lab 3: Finding Groups Using Recursion
Recursive algorithms for identifying and analyzing connected components in data structures.

### Lab 4: Searching and Sorting
Implementation and comparison of various searching and sorting algorithms (QuickSort, MergeSort, Binary Search).

### Lab 5: Spell Checker Using a Hash Table
Dictionary implementation using hash tables with efficient lookup and collision resolution.

### Lab 6: Finding the Closest Pair of Points
Computational geometry problem using divide-and-conquer algorithm for optimal point pair detection.

### Lab 7: Binary Search Tree
Complete BST implementation with insertion, deletion, and traversal operations.

### Lab 8: Storing a Sequence in a Binary Search Tree
Advanced BST operations for sequence storage and efficient retrieval.

### Lab 9: Using Recursion to Solve the N-Queens Problem
Classic backtracking problem demonstrating recursive problem-solving and constraint satisfaction.

### Lab 10: Using Breadth-First Search to Solve Puzzles
Graph traversal algorithms applied to puzzle-solving (sliding puzzles, path finding).

### Lab 11: Iterative Refinement and the Traveling Salesman Problem
Optimization algorithms and heuristics for solving NP-hard problems like TSP.
