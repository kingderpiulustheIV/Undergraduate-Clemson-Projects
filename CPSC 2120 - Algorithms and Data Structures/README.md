# CPSC 2120 - Algorithms and Data Structures

## Course Description
Advanced study of algorithms and data structures with emphasis on efficiency analysis and algorithmic problem-solving. Topics include advanced trees, graphs, hashing, heaps, sorting algorithms, algorithm design techniques, and complexity analysis.

## Learning Objectives
- Master advanced data structures (heaps, hash tables, graphs, trees)
- Analyze algorithm complexity using Big-O notation
- Implement efficient searching and sorting algorithms
- Apply algorithmic design patterns (greedy, divide-and-conquer, dynamic programming)
- Solve complex computational problems using appropriate data structures
- Understand graph algorithms and traversal techniques

## Projects and Assignments

### Min-Heap Implementation Using a Vector
Implement a min-heap data structure using vector-based storage, including insertion, deletion, and heapify operations.

### Queue Implementation Using a Circular Array
Efficient queue implementation using circular array technique to optimize space utilization.

### Single-Word Anagram Finder
String processing application using hash maps and efficient searching to find anagrams.

### Stack-based Postfix Expression Calculator
Expression evaluation using stack data structure to process postfix notation.

### Swapping Nodes in a Doubly Linked List
Advanced linked list operations demonstrating pointer manipulation and node management.

### Word Ladder
Graph-based puzzle solver using breadth-first search to find transformation paths between words.
