# CPSC 1020 - Computer Science II

## Course Description
Introduction to algorithms and data structures with an emphasis on problem solving and software design. Topics include object-oriented programming in C++, recursion, algorithm analysis, linked lists, stacks, queues, trees, and sorting algorithms.

## Learning Objectives
- Master object-oriented programming concepts using C++
- Understand and implement fundamental data structures (linked lists, stacks, queues, trees)
- Apply recursion to solve complex problems
- Analyze algorithm efficiency and complexity
- Develop problem-solving skills through programming challenges

## Projects and Assignments

### Cities
Working with data structures to manage and process city information.

### Programming Exam 1
Practical assessment of fundamental C++ concepts and data structure implementation.

### Programming Exam 2
Advanced problem-solving assessment covering recursion and complex data structures.

### Project 1: Isolation Self Assessment Tool
Implementation of game logic and AI for the Isolation board game.

### Project 2: Quiz Game
Interactive quiz application demonstrating file I/O and data management.

### Project 3: Tigergame
Comprehensive game development project integrating multiple data structures and algorithms. The game is a spin on the card game Blackjack.
